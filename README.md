# OnYourSide23

OnYourSide22 is an automated stock trading program that can work in your favor.

## Requirements

* Python 3.x
* PyQt5
* PyWin32
* requests

## Installation

1. Clone this repository to your local machine.
2. Install required packages using pip:

pip install -r requirements.txt
